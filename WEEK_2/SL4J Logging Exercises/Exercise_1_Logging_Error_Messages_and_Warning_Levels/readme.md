### Exercise 1 - Logging Error Messages and Warning Levels

\# Exercise 1 - Logging Error Messages and Warning Levels



\## Objective



Demonstrate logging of error and warning messages using the SLF4J API with Logback.



\## Technologies Used



\- Java 17

\- Maven

\- SLF4J 1.7.30

\- Logback Classic 1.2.3

\- IntelliJ IDEA



\## Project Structure



```

src

└── main

&#x20;   └── java

&#x20;       └── LoggingExample.java

```



\## Description



This project demonstrates how to configure SLF4J with Logback in a Maven project and log messages using different severity levels.



\## Log Levels Used



\- ERROR

\- WARN



\## Expected Output



```

ERROR LoggingExample - This is an error message

WARN  LoggingExample - This is a warning message

```
