### ORM Learn



\## Exercise



Spring Data JPA - Basic ORM Application



\## Objective



Develop a Spring Boot application that connects to a MySQL database using Spring Data JPA. The application stores and retrieves Country records using Hibernate ORM.



\## Technologies Used



\- Java 17

\- Spring Boot

\- Spring Data JPA

\- Hibernate

\- Maven

\- MySQL



\## Project Structure



```

src

&#x20;└── main

&#x20;     ├── java

&#x20;     │    └── com.cognizant.orm\_learn

&#x20;     │          ├── model

&#x20;     │          │     └── Country.java

&#x20;     │          ├── repository

&#x20;     │          │     └── CountryRepository.java

&#x20;     │          ├── service

&#x20;     │          │     └── CountryService.java

&#x20;     │          └── OrmLearnApplication.java

&#x20;     └── resources

&#x20;            └── application.properties

```



\## Features



\- Connects to MySQL database.

\- Uses Spring Data JPA Repository.

\- Inserts Country records.

\- Retrieves all Country records.

\- Displays records in the console.
