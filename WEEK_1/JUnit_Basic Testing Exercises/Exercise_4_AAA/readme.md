### Exercise 4 - Arrange-Act-Assert (AAA) Pattern



\# Exercise 4 - AAA Pattern, Setup and Teardown in JUnit



\## Objective

Implement the Arrange-Act-Assert (AAA) testing pattern using JUnit with setup and teardown methods.



\## Technologies Used

\- Java 17

\- Maven

\- JUnit 4.13.2

\- IntelliJ IDEA



\## Project Structure

```

src

├── main

│   └── java

│       └── MathOperations.java

└── test

&#x20;   └── java

&#x20;       └── AAATest.java

```



\## Features

\- Arrange-Act-Assert (AAA) Pattern

\- @Before Annotation

\- @After Annotation

\- Multiple Test Cases



\## Description

This project demonstrates how to organize unit tests using the AAA pattern. It also shows how to initialize objects before each test using `@Before` and clean up resources after each test using `@After`.



\## Tests Performed

\- Addition Test

\- Multiplication Test
